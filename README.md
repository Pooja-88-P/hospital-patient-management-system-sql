# Hospital Patient Management System (SQL Project)

## Project Overview
A Hospital Management System database designed using MySQL to manage:

- Patients
- Doctors
- Appointments
- Treatments
- Prescriptions

## Database Design

### Tables
1. Patients
2. Doctors
3. Appointments
4. Treatments
5. Prescriptions

## SQL Concepts Used

- DDL Commands
- DML Commands
- Primary Keys
- Foreign Keys
- INNER JOIN
- GROUP BY
- HAVING
- Aggregate Functions

## Advanced Queries

- Most Visited Doctor
- Top 3 Diagnoses
- Patients with Multiple Visits
- Doctor Workload Analysis
- Appointment Trends
- Complete Hospital Report
